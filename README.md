# SC Cargo Hauler

A cargo hauling run tracker for Star Citizen. Stack contracts onto a ship, monitor SCU usage, and track payout efficiency across runs.

## Features

- **Ship selection** — pick from known ships with preset SCU limits or enter a custom capacity
- **Contract stacking** — add multiple contracts with commodity, SCU, payout, pickup/drop-off locations, and distance
- **Cargo alerts** — warns when approaching or exceeding ship cargo capacity
- **Run metrics** — total SCU, remaining capacity, total payout, UEC/SCU, UEC/km, unique drop-off count
- **Run history** — completed runs are saved locally with expandable contract details
- **Lifetime stats** — total UEC earned, total SCU hauled, average UEC/run, best run
- **Auto-save** — current run persists across page reloads

## Usage

Open `index.html` in any browser. No build step or server required.
